# calcu_perimetro
Calcula el perímetro de diferentes figuras geométricas:  Cuadrado, rectángulo, trapecio, circulo, polígono regular, triángulo equilátero, triángulo isósceles, triángulo escaleno y rectángulo. Guarda el resultado en un archivo JSON, permite buscar en el archivo por figura. Utiliza la salida por consola formateada con la librería rich. Python 3.8.10.
